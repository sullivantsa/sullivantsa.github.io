About This Website

The Sullivan TSA Chapter created this website for a 2016 webmaster entry for state competition. The website gives information ranging from chapter info to contacting our chapter to our 2016 Design Brief.

How was this website created?

This website was created using modern technology and innovation. This website was created by using true programming and the lasted version of HTML, HTML5. HTML5 outlines what the website says, while the "looks" of the website was customized using CSS3. Any actions and "neat" features were created using JavaScript. The underlying makeup of the website is unique versus websites created using framework systems because pure programming allows for various customizations. Wordpress and other framework systems are simple drag and drop programs that are for simple websites and have absolutely no original content. The program itself does all the hard work.
