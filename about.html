About This Website

The Sullivan TSA Chapter created this website for a 2016 webmaster entry for state competition. The website gives information ranging from chapter info to contacting our chapter to our 2016 Design Brief.

How was this website created?

This website was created using modern technology and innovation. Using true programming, I have been able to create an “original” website. Not one that uses a framework system (Wordpress, Drupal, etc.) but real code. The basis of this website is built upon HTML (text), CSS (the “looks”), and JavaScript (the animations). The use of real programming versus the modern-day version of a “webmaster” gives many advantages. For example, the use of code allows the user to customize the website to the extent in which he/she wants. Using a framework system refrains the user from the in-depth customization.
